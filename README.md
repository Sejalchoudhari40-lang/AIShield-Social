# рҹӣЎпёҸ AIShield

> **AI-Powered Content Moderation Platform**

AIShield is a full-stack AI-powered content moderation platform built using **React, TypeScript, FastAPI, SQLAlchemy, PyTorch, and Transformers**. It analyzes uploaded images and text using Artificial Intelligence while providing a secure authentication system, analytics dashboard, history management, and user profile features.
